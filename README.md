# Function Selector

Esta extensión permite seleccionar funciones rápidamente en tu código JavaScript desde VS Code.

## Features

- Lista todas las funciones definidas en un archivo.
- Permite saltar directamente a la definición.
- Soporte para archivos `.js`.

## Requirements

Ninguna dependencia adicional.
